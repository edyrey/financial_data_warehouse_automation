## Google Sheets Setup

1. Create a Google Sheet (acts as the warehouse)
2. Add sheets:
   - GL
   - Final
   - Missing_GL_Mapping
3. Open Extensions → Apps Script
4. Paste Code.gs
5. Enable Advanced Drive API (v3)
6. Paste your Monthly_Inputs folder ID
7. Reload the sheet
8. Run: NARPM Warehouse → Run Monthly Update
