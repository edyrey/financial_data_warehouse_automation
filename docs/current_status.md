# Current Status

This repository is being restructured to reflect the project's current state more accurately than the earlier public version.

The original public repository emphasized the Google Suite implementation because that was the main production workflow at the time. The current project direction is different:

- the Google Suite phase remains an important part of the project history
- the Python implementation is now the primary focus
- the current rebuild is aimed at a local data warehouse workflow with stronger documentation and cleaner version control

This repo is therefore intended to show both the historical path of the project and the present development direction.
